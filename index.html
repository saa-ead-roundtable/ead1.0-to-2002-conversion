EAD 1.0 to EAD 2002 Conversion Package

Download .zip Download .tar.gz View on GitHub

This package converts valid EAD 1.0 instances to EAD 2002 instances and optionally produces a report of the changes made during conversion. It also parses ISO 639-2 attribute values, writing out language code and language name, in either French or English and removes all deprecated and obsolete elements (replacing where appropriate).

This package is provided by the Encoded Archival Description Roundtable (EADRT) of the Society of the American Archivists. The EAD Roundtable cannot provide support for this package.

More documentation can be found online at https://github.com/saa-ead-roundtable/ead1.0-to-2002-conversion/blob/master/readme.txt
